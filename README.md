## 📖 Project Overview
This project performs Exploratory Data Analysis (EDA) on a movie dataset to uncover trends in content distribution, genre popularity, and audience engagement.

The goal is to extract meaningful insights that can help understand viewing patterns and support data-driven decisions in the entertainment industry.

---

## 🎯 Objectives
- Analyze distribution of content across years  
- Identify most popular genres  
- Study trends in movie releases over time  
- Analyze popularity vs vote count  
- Categorize content based on ratings  
- Extract business insights from data  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📂 Dataset Features
- Release Date  
- Title  
- Popularity  
- Vote Count  
- Vote Average  
- Genre  

---

## 🔧 Data Preprocessing
- Converted `Release_Date` to year format  
- Removed unnecessary columns  
- Handled categorical transformations  
- Categorized `Vote_Average` into:
  - Not Popular  
  - Below Average  
  - Average  
  - Popular  
- Split and exploded multi-genre values  

---

## 📊 Key Analysis Performed
- Most frequent genre identification  
- Distribution of content over years  
- Popularity vs Vote Count analysis  
- Genre-wise insights  
- Data visualization using graphs  

---

## 📈 Key Insights
- Drama is the most frequent genre  
- Majority of content is released after 2000  
- Popularity has strong outliers  
- Balanced distribution across rating categories
